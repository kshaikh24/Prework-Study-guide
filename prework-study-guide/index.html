HTML

  • HTML stands for HyperText Markup Language.
  • Html is like the underlying structure of a house.
  • It is used to create the structure of web pages on the internet.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • H1 - H6 are used to establish visual hierarchy, and are useful for people with accessibility tools.
